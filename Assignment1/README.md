# React Form Application

This is a simple React-based form project demonstrating form handling, validation, and conditional rendering.  
The submit button becomes active only when all required fields are filled correctly.

---

## Commands to Run

```bash
# Clone the repository
git clone https://github.com/SitaDasari/upzenix-assignments.git

# Go to project folder
cd upzenix-assignments/Assignment1

# Install dependencies
npm install
npm install react-router-dom

# Start the development server
npm start
```
